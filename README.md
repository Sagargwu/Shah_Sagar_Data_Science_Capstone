This repository holds my Spring 2026 Data Science Capstone project for DATS 6501 at The George Washington University. The project is centered on developing and implementing a deep learning model based lane detection system for driver assistance systems. The goal is to correctly detect road lane boundaries from camera images and driving videos using a Fully Convolutional Neural Network (FCN) for pixel level image segmentation.

In contrast to other computer vision approaches that traditionally depend on rule based definitions such as edge detection or color-based thresholding, this system takes a data-driven approach. The system learns spatial and contextual features directly from data, which enhances the system's ability to handle real-world scenarios such as shadows, curved roads, deteriorated lane markings, and different lighting conditions.

The project comprises an entire end-to-end machine learning process, from data preprocessing and model training and evaluation to video inference. Once deployed, the trained model produces binary masks of the lanes, which are superimposed on original video frames to generate annotated driving videos. This capstone project showcases the real world application of deep learning in computer vision and the integration of model development, system design, and real-world visualization in a Git-based workflow.
